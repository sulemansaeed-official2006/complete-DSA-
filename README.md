1. Start the Local Server
Launch the XAMPP Control Panel.

Start both the Apache and MySQL modules.

2. Configure the Database
Open phpMyAdmin via the XAMPP Control Panel (click Admin next to MySQL) or go to http://localhost/phpmyadmin.

Create a new database matching your project's configuration.

Execute the required SQL queries to set up the schema for the Login and Signup pages.

3. Deploy Project Files
Navigate to your XAMPP installation directory (e.g., C:\xampp\htdocs).

Paste your project folder directly into the htdocs directory.

4. Run the Application
Open your web browser.

Navigate to the following address (replace project-name with your actual folder name):

HTTP
http://localhost/project-name
